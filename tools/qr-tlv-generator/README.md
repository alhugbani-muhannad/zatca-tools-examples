# 🏷️ توليد QR المرحلة الأولى (TLV Generator)

توليد رمز QR متوافق مع **المرحلة الأولى** من الفوترة الإلكترونية السعودية: خمسة وسوم TLV (اسم البائع، الرقم الضريبي، طابع الوقت، الإجمالي، الضريبة) مشفّرة base64 حسب مواصفات ZATCA.

## الاستخدام

```bash
python generate.py "مؤسسة الأمل التجارية" 310122393500003 "2026-07-19T14:30:00Z" 115.00 15.00
```

الناتج سلسلة base64 جاهزة — حوّلها لصورة بأي مكتبة QR:

```bash
pip install qrcode
python -c "import qrcode, subprocess; qrcode.make(subprocess.run(['python','generate.py','مؤسسة الأمل','310122393500003','2026-07-19T14:30:00Z','115.00','15.00'],capture_output=True,text=True).stdout.strip()).save('qr.png')"
```

## ⚠️ مهم — المرحلة الثانية

رموز **المرحلة الثانية** تتطلب وسومًا إضافية (6–9): تجزئة XML، توقيع ECDSA، المفتاح العام، وتوقيع الشهادة — ولا يمكن توليدها إلا بشهادة ختم تشفير معتمدة من الهيئة عبر عملية Onboarding كاملة (CSR ← CCSID ← فحوص التوافق ← PCSID).
